# WxReader-fish

基于[Pake](https://github.com/tw93/Pake)开发的一款微信读书**摸鱼**客户端

## 支持功能

- 透明窗口阅读
- 老板键(Alt+V)
- 快捷键上下页(Alt+Z,Alt+C)
- 窗口状态保存
- 微信阅读网页版的所有功能

## 操作提示

移动窗口->侧边面板空白处拖住可以移动窗口

换章节,修改字体字号等 -> 拉宽窗口,微信阅读会自适应显示操作按钮在右侧
